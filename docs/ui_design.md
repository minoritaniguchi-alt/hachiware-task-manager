## 1. デザインコンセプト

Koto Note のデザインコンセプト。ハチワレをモチーフにした「モチーフ × アイテム」のミニマルデザイン。
**キャラクターの絵文字には依存せず、幾何学的モチーフとアイテムアイコンで世界観を表現する。**

- **モチーフ中心**: ハチワレの耳（△△）を幾何学SVGで随所に配置
- **ミニマル北欧風**: 広い余白・薄いボーダー・高コントラストでないテキスト
- **アイテムアイコン**: ハチワレの好きなもの（🍜 📷 🎸 🎀）をUIに採用
- **実用性**: ステータス管理・フィルター・アーカイブを備えた本格タスク管理

## 4. 猫モチーフ（幾何学SVG）

### 4-1. EarDecor — アウトライン耳（ヘッダー・トースト用）

2本の△△ アウトライン線で耳を表現。ヘッダーロゴ左上・入力カード右上・セクション見出し・トーストに配置。

```
viewBox: "0 0 30 16"
左耳: M1,15 L8,1 L15,15   (stroke)
右耳: M15,15 L22,1 L29,15 (stroke)
```

### 4-2. EarCheckbox — タスク完了トグル

| 状態 | 表示 | 色 |
|------|------|----|
| 未完了 | アウトライン三角×2 | `stroke="#A2C2D0"` |
| 完了 | 塗りつぶし三角×2 | `fill="#4A9E68"` |

### 4-3. CatEarDecor — ダッシュボードカード用猫耳（絶対配置）

カードの枠線と一体化した塗りつぶし三角形。カードWrapper に `pt-5`（20px）を付与し、
耳SVG（高さ22px）を `top-0` に配置することで、耳底辺の2pxがカード上辺と重なり
「枠から耳が生えて見える」視覚効果を実現。

```
viewBox: "0 0 28 22"
パス: M1,22 L14,1 L27,22 Z  (fill: カード枠線と同色)
```

| カード | 耳位置 | CSS クラス |
|-------|-------|----------|
| ルーチン業務（青） | 左上 | `absolute top-0 left-4` |
| 臨時対応（ピンク） | 上中央 | `absolute top-0 left-1/2 -translate-x-1/2` |
| 予定（緑） | 右上 | `absolute top-0 right-4` |

### 4-4. CatTailDecor — ダッシュボードカード用猫尻尾（絶対配置）

カードの側面ボーダーと同色のSVG曲線（C字カーブ）。`overflow: visible` のカード外へ
`translate-x-full` / `-translate-x-full` で完全に押し出し、「枠から尻尾が伸びる」表現。

```
viewBox: "0 0 20 48"
右尻尾: M0,6 Q18,6 18,22 Q18,40 0,44   (stroke, strokeWidth: 2.8)
左尻尾: M20,6 Q2,6 2,22 Q2,40 20,44   (stroke, strokeWidth: 2.8)
```

| カード | 尻尾位置 | CSS クラス |
|-------|---------|----------|
| ルーチン業務（青） | 右下 | `absolute right-0 bottom-8 translate-x-full` |
| 臨時対応（ピンク） | なし | — |
| 予定（緑） | 左下 | `absolute left-0 bottom-8 -translate-x-full` |

### 4-5. 折りたたみ機能との共存

`overflow-hidden` をカード本体から外し、コンテンツの折りたたみは
CSSアニメーションではなく**条件レンダリング（`{open && ...}`）** で実現することで、
尻尾SVGがカード外へはみ出しても機能が正常動作することを保証する。

## 5. アイテムアイコン

ハチワレの好きなものをカテゴリアイコン・Empty State に採用。

| アイテム | 絵文字 | 使用箇所 |
|---------|-------|---------|
| ラーメン（チャリメラ） | 🍜 | ルーチン業務カードアイコン |
| カメラ（一眼レフ） | 📷 | 臨時対応カードアイコン |
| ギター | 🎸 | 予定カードアイコン |
| ほめられリボン | 🎀 | Empty State（タスクなし時） |

### Empty State 表示

```
🍜  📷  🎸  🎀

タスクはありません
```

## 6. タイポグラフィ

- **フォント**: Noto Sans JP（Google Fonts）
- **ウェイト**: 400（通常）/ 500（ラベル・見出し）/ 700（ボタン）
- **セクション見出し**: `text-xs tracking-widest uppercase` — 北欧ミニマル風
- **本文**: `text-sm`
- **バッジ・補助テキスト**: `text-xs`

## 7. テキストガイドライン

キャラクター固有のセリフは使用しない。事務的でシンプルな日本語に統一。

| 旧テキスト（廃止） | 新テキスト |
|-----------------|----------|
| 「タスクがないよ！のんびりしよ💙」 | 「タスクはありません」 |
| 「まだ何もないよ🐱」 | 「項目なし」 |
| 「完了したタスクたち💙 エライッ！」 | 「完了済みのタスク」 |
| 「思い出アーカイブ ✅」 | 「完了済み」 |
| 「リストに戻したよ🐱」 | 「リストに戻しました」 |
| 「新しいタスクを追加しよう！」 | 「新しいタスクを入力」 |
