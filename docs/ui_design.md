## 1. デザインコンセプト

ちいかわの「ハチワレ」の世界観を大切にしたデザイン。

- **やさしさ**: くすみカラーと丸みのあるシェイプで、見るだけでほっとする
- **かわいらしさ**: 🐱 💙 などの絵文字と、「えらい！」「のんびりしよ」などの言葉遣い
- **実用性**: スプレッドシートより見やすく、ステータスが一目でわかるバッジ設計

## 3. タイポグラフィ

- **フォント**: Noto Sans JP（Google Fonts）
- **ウェイト**: 300（細め本文）/ 400（通常）/ 500（中程度・ラベル）/ 700（見出し・ボタン）
- **本文サイズ**: 14px（sm: text-sm）
- **見出しサイズ**: 18px（lg: text-lg）
- **バッジ・ラベル**: 12px（xs: text-xs）

## 5. コンポーネント設計

### 5-1. StatusBadge
- クリックで展開するドロップダウン形式
- ステータスごとに配色が異なるピル型バッジ
- 現在選択中のステータスにはリングハイライト

### 5-2. TaskInputForm
- テキストフィールドにフォーカスすると詳細欄が展開（アニメーション付き）
- メモ・ステータス・優先度・期限を追加設定可能
- 追加ボタンはハチワレブルーのソリッドボタン

### 5-3. DashboardCard
- 2px のカテゴリ別カラーボーダー
- グラデーション背景（カテゴリカラーの薄いグラデ）
- タイトル行をクリックで折りたたみ可能（ChevronUp/Down）

### 5-4. TaskRow
- ホバーでクリーム色の背景
- グリップアイコン（左端）、操作ボタン（右端・ホバー時に表示）
- Done タスクは取り消し線 + 透明度 50%

### 5-5. Toast
- 右下に固定表示（fixed bottom-6 right-6）
- 2.5秒後に自動消去
- フェードイン アニメーション
