## 1. デザインコンセプト

ハチワレをモチーフにした「モチーフ × アイテム」のミニマルデザイン。
**キャラクターの絵文字には依存せず、幾何学的モチーフとアイテムアイコンで世界観を表現する。**

- **モチーフ中心**: ハチワレの耳（△△）を幾何学SVGで随所に配置
- **ミニマル北欧風**: 広い余白・薄いボーダー・高コントラストでないテキスト
- **アイテムアイコン**: ハチワレの好きなもの（🍜 📷 🎸 🎀）をUIに採用
- **実用性**: ステータス管理・フィルター・アーカイブを備えた本格タスク管理

## 4. 耳モチーフ（幾何学デザイン）

ハチワレの耳を2つの三角形（△△）として抽象化し、SVGで実装する。

### 4-1. EarDecor — 装飾パーツ

```jsx
// viewBox: "0 0 30 16"
// 左耳: M1,15 L8,1 L15,15
// 右耳: M15,15 L22,1 L29,15
```

**配置箇所:**
- ヘッダーロゴ左上（`size=10, opacity=0.5`）
- タスク入力カード右上（`size=13, opacity=0.3`）
- ダッシュボードセクション見出し（`size=10, opacity=0.6`）
- トースト通知内（`size=12, opacity=0.8`）

### 4-2. EarCheckbox — タスク完了トグル

タスクリストの左端に配置するインタラクティブな耳型チェックボックス。

| 状態 | 表示 | 色 |
|------|------|----|
| 未完了 | アウトライン三角×2 | `stroke="#A2C2D0"` |
| 完了 | 塗りつぶし三角×2 | `fill="#4A9E68"` |

クリックで done ↔ doing をトグルする。

## 5. アイテムアイコン

ハチワレの好きなものをカテゴリアイコン・Empty State に採用。

| アイテム | 絵文字 | 使用箇所 |
|---------|-------|---------|
| ラーメン（チャリメラ） | 🍜 | ルーチン業務カードアイコン |
| カメラ（一眼レフ） | 📷 | 臨時対応カードアイコン |
| ギター | 🎸 | 予定カードアイコン |
| ほめられリボン | 🎀 | Empty State（タスクなし時） |

### Empty State 表示

```
🍜  📷  🎸  🎀

タスクはありません
```

## 6. タイポグラフィ

- **フォント**: Noto Sans JP（Google Fonts）
- **ウェイト**: 400（通常）/ 500（ラベル・見出し）/ 700（ボタン）
- **セクション見出し**: `text-xs tracking-widest uppercase` — 北欧ミニマル風
- **本文**: `text-sm`
- **バッジ・補助テキスト**: `text-xs`

---

## 7. テキストガイドライン

キャラクター固有のセリフは使用しない。事務的でシンプルな日本語に統一。

| 旧テキスト（廃止） | 新テキスト |
|-----------------|----------|
| 「タスクがないよ！のんびりしよ💙」 | 「タスクはありません」 |
| 「まだ何もないよ🐱」 | 「項目なし」 |
| 「完了したタスクたち💙 エライッ！」 | 「完了済みのタスク」 |
| 「思い出アーカイブ ✅」 | 「完了済み」 |
| 「リストに戻したよ🐱」 | 「リストに戻しました」 |
| 「新しいタスクを追加しよう！」 | 「新しいタスクを入力」 |
